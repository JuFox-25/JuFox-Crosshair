## ✨ Features

- **Multi-Shape** — Combine multiple crosshair shapes simultaneously (Cross + Circle, X + Dot, etc.)
- **Fully Customizable** — Color, size, opacity, outline, rotation, stretch
- **RMB Actions** — Hide lines/shapes when right-clicking (aiming/ADS)
- **ADS Crosshair** — Separate crosshair that appears only when right-clicking
- **RMB Effects** — Change opacity, size and color when aiming
- **Multi-Monitor** — Choose which monitor the crosshair appears on
- **Profiles** — Save and load unlimited custom profiles
- **Live Preview** — See changes before applying
- **Works in any game** — Transparent overlay, always on top
- **Runs in background** — System tray icon, starts minimized

## 💾 Installation

### Easy Install (Recommended)
1. Download `JuFoxCrosshair-Setup.exe` from [Releases](https://github.com/JuFox/JuFox-Crosshair/releases)
2. Run the installer
3. Done! Desktop shortcut is created automatically

### Manual Install
**Requirements:** [Node.js LTS](https://nodejs.org)

```bash
git clone https://github.com/JuFox/JuFox-Crosshair.git
cd JuFox-Crosshair
npm install
npm start
```

## 🎮 How to Use

| Action | Shortcut |
|--------|----------|
| Open Settings | `Alt+C` |
| Toggle Overlay | `Alt+X` |
| Quit App | Right-click tray icon → Quit |

### RMB Feature
1. Open Settings → **RMB / ADS** tab
2. Select which lines/shapes to hide when right-clicking
3. Optionally set up an ADS crosshair
4. Click **Apply**
5. Now when you right-click in-game, your crosshair changes!
